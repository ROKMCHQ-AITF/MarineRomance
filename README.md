모델 선택 근거
ResNet18Audio
ImageNet으로 사전학습된 CNN 계열 모델로, 멜 스펙트로그램을 2D 이미지로 처리한다. ESC-50처럼 레이블이 명확하고 클립 길이가 고정된(5초) 데이터셋에서 CNN은 주파수-시간 패턴을 효과적으로 포착한다. 파라미터 수가 약 11M으로 데이터 500~5K 구간에 적합하며, 사전학습 가중치 덕분에 적은 데이터로도 안정적인 성능을 낸다. 구조가 단순하고 학습이 빠르기 때문에 파이프라인 검증 및 베이스라인 성능 기준점으로 선택했다.
CRNNAudio
CNN과 Bidirectional LSTM을 결합한 구조로, CNN이 주파수 축의 로컬 패턴을 추출하고 LSTM이 시간 축의 순차 패턴을 처리한다. ESC-50의 환경음 중 일부(빗소리, 파도 소리, 기계음 등)는 시간에 따라 변화하는 패턴을 가지기 때문에, 정적인 CNN만으로는 놓칠 수 있는 시간적 변화를 LSTM이 보완한다. 파라미터 수가 약 2.7M으로 세 모델 중 가장 가볍고, 사전학습 없이도 학습 가능하다는 점에서 네트워크 환경이 제한된 상황에서도 안정적으로 사용할 수 있다.
EfficientNetAudio
EfficientNet-B0는 같은 파라미터 수 대비 ResNet보다 높은 성능을 내도록 설계된 경량 CNN으로, Compound Scaling을 통해 깊이/너비/해상도를 동시에 최적화한다. BirdCLEF 같은 오디오 분류 대회 상위권에서 멜 스펙트로그램 + EfficientNet 조합이 표준처럼 사용되고 있으며, ImageNet 사전학습 가중치를 활용해 적은 데이터에서도 높은 성능을 기대할 수 있다. ResNet18보다 파라미터 효율이 좋아 동일한 학습 비용으로 더 나은 성능을 낼 가능성이 높다.

데이터 분할 근거
ESC-50은 파일명에 src_file 컬럼이 포함되어 있으며, 동일한 원본 녹음에서 잘라낸 여러 클립이 존재한다. 이를 세그먼트 단위로 무작위 분할하면 같은 원본에서 나온 클립이 학습과 테스트에 동시에 포함되어 데이터 누수(Data Leakage)가 발생하고 성능이 실제보다 과대평가된다. ESC-50은 이 문제를 방지하기 위해 설계 단계에서 이미 fold 컬럼을 제공하며, 동일 원본의 클립을 같은 fold에 묶어두었다. 따라서 fold 컬럼을 기준으로 분할하는 것이 GroupKFold와 동일한 효과를 내며, 데이터 누수 없이 신뢰할 수 있는 성능 평가가 가능하다.

손실 함수 근거
CrossEntropy에 Label Smoothing(ε=0.1)을 적용했다. ESC-50의 50개 클래스 중 일부는 소리가 유사해 혼동되기 쉽다(예: rain vs sea_waves, dog vs cat). 일반 CrossEntropy는 정답 클래스에 확률 1.0을 목표로 학습하기 때문에 모델이 과도하게 확신하는 방향으로 과적합될 수 있다. Label Smoothing은 정답 레이블을 1.0 대신 0.9로, 나머지 클래스에 0.1을 균등 분배해 모델이 지나치게 확신하지 않도록 유도하며, Transformer 계열 모델에서도 표준으로 사용된다.

옵티마이저 근거
AdamW를 선택했다. Adam은 각 파라미터마다 개별 학습률을 자동으로 조정해 수렴이 빠르고 안정적이지만, Weight Decay가 기울기 스케일링과 결합되어 정규화 효과가 제대로 작동하지 않는 문제가 있다. AdamW는 Weight Decay를 기울기 업데이트와 분리(decoupled)해 정규화 효과를 올바르게 적용하며, Transformer 계열 및 파인튜닝의 표준 옵티마이저로 자리잡았다. 학습률은 1e-3으로 시작하며, 사전학습 모델 파인튜닝 시에는 1e-4 이하로 낮추는 것을 권장한다.

학습률 스케줄러 근거
Cosine Annealing을 적용했다. 고정 학습률은 학습 초반에는 수렴이 느리고 후반에는 최적점을 지나치는 문제가 있다. Cosine Annealing은 학습률을 코사인 곡선을 따라 부드럽게 감소시켜, 학습 초반에는 빠른 수렴을 유도하고 후반에는 세밀한 조정이 가능하게 한다. Step Decay처럼 감소 시점을 수동으로 설정할 필요 없이 자동으로 작동하며, AST, Swin Transformer 등 최신 오디오 분류 모델에서 표준으로 채택하고 있다.

정규화 근거
세 가지 정규화 기법을 동시에 적용했으며, 각각 작동하는 위치가 달라 서로 겹치지 않는다.
**Dropout(0.3)**은 학습 중 뉴런을 랜덤하게 비활성화해 특정 뉴런에 의존하는 것을 방지한다. 모델 내부에서 작동하며 과적합을 직접 억제한다.
**Weight Decay(0.01)**는 가중치 크기에 패널티를 부여해 가중치가 지나치게 커지는 것을 방지한다. AdamW에서 기울기 업데이트와 분리되어 적용되므로 올바르게 작동한다.
**Gradient Clipping(max_norm=1.0)**은 역전파 시 기울기가 폭발적으로 커지는 것을 방지한다. 기울기의 크기(norm)가 임계값을 초과하면 방향은 유지한 채 크기만 줄인다. Transformer 계열과 LSTM에서 특히 중요하며, ResNet에서도 안정성을 위해 적용했다.

전처리 근거
멜 스펙트로그램(128 mel bins, n_fft=1024, hop_length=512)을 선택했다. 사람의 청각 특성에 맞게 주파수 축을 로그 스케일로 변환하며, 딥러닝 생태계에서 가장 많은 사전학습 모델이 이 형식을 기반으로 한다. Log를 추가로 적용해 넓은 에너지 범위를 압축하고 작은 차이를 더 잘 구별하도록 했다. 출력을 3채널로 복제한 것은 ImageNet으로 사전학습된 모델이 RGB 3채널 입력을 기대하기 때문이며, 이 방식이 음향 분류 대회에서 표준으로 사용된다.
학습 시에만 SpecAugment(FrequencyMasking, TimeMasking)를 적용해 과적합을 추가로 방지했다. SpecAugment는 멜 스펙트로그램의 일부 주파수 대역과 시간 구간을 무작위로 마스킹해 모델이 특정 패턴에 의존하지 않도록 강제하며, 음향 분류 분야에서 가장 검증된 데이터 증강 기법이다.
